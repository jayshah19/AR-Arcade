Detailed Description

This class contains the variable and method to spawn BeerCup GameObject on a detected AR Plane for the BeerPong game in ARArcade Project.

Checks if the beer cup is placed. If it is not placed then the code below will be executed, otherwise the function will return at this step.

Checks wether the user has touched the screen where the plane is detected. If true then the BeerCup GameObject will be instantiated and the first throwable ball will be spawned.
