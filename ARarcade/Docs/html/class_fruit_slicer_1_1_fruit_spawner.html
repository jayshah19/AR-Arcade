IEnumerator FruitSlicer.FruitSpawner.spawnFruits ( )
private

This method spawns random fruits from the fruitPrefabs array at random spawn points from spawnPoints array.

Furthermore, we apply force to the spawned fruits in the upward direction, wait for a 4 seconds and destroy the fruit object if player was not able to slice the fruit.
